01 — Features

Built different.

01
SRS Rotation
Full Super Rotation System with wall kicks — exactly like guideline Tetris. No shortcuts, no compromises.
02
Live Audio Engine
Korobeiniki synthesized in real-time via Web Audio API. No MP3s — pure sound design.
03
Ghost Piece
Ghost projection and 7-bag randomizer for fair, strategic play every session.
04
Particle FX
Line clears trigger particle explosions, screen shake, and full-screen flash effects that make every clear feel earned.

04 — About

Built by
Davide.

Hi — I'm Davide, a student and AI enthusiast. I built BLOKFALL completely from scratch, for free, as a personal project to push what's possible in the browser.

No team. No budget. Just curiosity, code, and a lot of late nights.

Every piece of this — the SRS rotation engine, the synthesized audio, the particle effects, the UI — was hand-crafted with the help of AI tools that made it possible to build something this polished as a solo student.
